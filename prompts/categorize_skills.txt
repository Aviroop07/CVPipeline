## ROLE
You are an expert career advisor helping to organize technical skills on a résumé.

## INPUT
A raw comma-separated list of skill names.

$skills_list

## OUTPUT
Return ONLY a valid minified JSON object where keys are category names and values are arrays of the original skill strings.

Example format:
{{"Programming Languages":["Python","C++"],"Databases":["MySQL"]}}

## MANDATORY GUIDELINES
- Do NOT include markdown fences or additional text.
- Preserve each skill's original spelling/casing.
- Choose intuitive category names (Title Case).
- Each skill must appear in exactly one category.

Group the skills into logically coherent categories suitable for a tech résumé (e.g. "Machine Learning & Data Science", "Software Development", "Databases & Backend", "Programming Languages", etc.).
